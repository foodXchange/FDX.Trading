# FoodXchange User Guide

## Getting Started

### 1. Login
- Navigate to http://localhost:3000
- Use your credentials or demo account
- Dashboard loads automatically

### 2. Dashboard Overview
- **Metrics**: View supplier stats
- **Recent Activity**: Latest communications
- **AI Insights**: Smart recommendations

## Key Features

### Supplier Management
1. **View Suppliers**
   - Click "Suppliers" in sidebar
   - Search by name, product, or location
   - Filter by status or rating

2. **Add Supplier**
   - Click "Add Supplier" button
   - Fill in required fields
   - Import bulk via CSV

### Email Analysis
1. **Analyze Email**
   - Go to "Email Analyzer"
   - Paste supplier email
   - Click "Analyze"
   - View AI classification

2. **Automated Responses**
   - Select suggested action
   - Customize if needed
   - Send directly

### Price Analytics
1. **View Predictions**
   - Navigate to "Analytics"
   - Select product
   - View 30-day forecast

2. **Set Alerts**
   - Click "Price Alerts"
   - Set target prices
   - Get notifications

### Campaigns
1. **Create Campaign**
   - Go to "Campaigns"
   - Select suppliers
   - Choose template
   - Schedule sending

## Tips & Best Practices
- Import suppliers in batches of 1000
- Run email analysis daily
- Review AI insights weekly
- Export reports monthly
