# Email CRM - Quick Reference Card

## 🚀 Daily 5-Minute Workflow

### Morning (2 min)
```
1. Open /email → Check numbers
2. Any "Interested"? → Respond now
3. Note open rate → Plan outreach
```

### Send Campaign (2 min)
```
1. Click "✉️ Compose"
2. Select suppliers (Ctrl+click)
3. Type products → "🤖 Generate"
4. "📤 Send" → Done!
```

### Check Pipeline (1 min)
```
/email/pipeline → See 4 columns:
🆕 New → 📤 Sent → 👁️ Opened → ✅ Interested
```

---

## 📊 What Numbers Mean

| Metric | Bad | OK | Good | Great |
|--------|-----|-----|------|-------|
| Open Rate | <30% | 30-50% | 50-70% | >70% |
| Click Rate | <10% | 10-20% | 20-30% | >30% |
| Response | <2% | 2-5% | 5-10% | >10% |

---

## ✉️ Email Templates

### New Inquiry
```
Products: [specific items you want]
AI writes: Introduction + Interest + Questions
Length: ~100 words
```

### Follow-up
```
References: Previous email
AI writes: Gentle reminder + Value prop
Length: ~75 words
```

## 🎯 Quick Actions

| See This | Do This |
|----------|---------|
| Supplier opened 2+ days ago | Send follow-up |
| New response | Read → Respond same day |
| Low open rate | Check subject/timing |
| "Interested" status | Call or detailed email |

## ⚡ Keyboard Shortcuts

- **Select Multiple**: `Ctrl+Click` (Cmd on Mac)
- **Select Range**: Click first, `Shift+Click` last
- **Cancel**: `Esc`
- **Send**: `Enter` (in forms)

## 🚦 Status Guide

- **Gray Badge** = Sent, not opened
- **Blue Badge** = Opened email
- **Green Badge** = Clicked link
- **Gold Badge** = Responded

---

## 💡 Pro Tips

1. **Best Time**: Tue-Thu, 9-11 AM
2. **Batch Size**: 20-30 per day
3. **Follow-up**: After 3 days
4. **Subject**: Keep automatic ones

---

## 🆘 Quick Fixes

**No opens?** → Check spam, try afternoon send
**No responses?** → More specific products
**Too many responses?** → Reduce daily sends
