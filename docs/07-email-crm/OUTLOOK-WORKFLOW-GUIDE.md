## 🎯 Quick Start with Outlook

### Your Daily Setup

1. **Open Two Windows Side-by-Side**
```
┌─────────────────────────┬─────────────────────────┐
│   FoodXchange Email     │    Microsoft Outlook    │
│   localhost:9000/email  │    Your Email Client    │
│                         │                         │
│  [Generate Emails Here] │   [Send Emails Here]   │
└─────────────────────────┴─────────────────────────┘
```

2. **Keyboard Shortcuts to Remember**
- `Windows + Left Arrow`: Snap FoodXchange to left
- `Windows + Right Arrow`: Snap Outlook to right
- `Ctrl + A`: Select all text
- `Ctrl + C`: Copy
- `Ctrl + V`: Paste

## 📧 Step-by-Step Workflow

### Morning Routine (15 minutes total)

#### Step 1: Generate First Batch (2 minutes)
1. In FoodXchange, click "✉️ Compose Email"
2. Select 5 suppliers (same country is easier)
3. Type specific products: `organic olive oil, 1L bottles`
4. Click "🤖 Generate with AI"

#### Step 2: Send via Outlook (3 minutes)
1. Copy the generated email (Ctrl+A, Ctrl+C)
2. In Outlook: `New Email` (Ctrl+N)
3. Click "To:" and add supplier emails
4. Paste email content (Ctrl+V)
5. Adjust greeting if you know contact name
6. Click Send (Alt+S)

#### Step 3: Repeat (10 minutes for 20 emails)
- Generate 5 emails → Copy/Send → Repeat
- Total: 4 batches = 20 personalized emails

## 💡 Outlook Pro Tips

### 1. Create Supplier Contact Groups

**One-time setup:**
1. In Outlook: People → New Contact Group
2. Name: "Italian Food Suppliers"
3. Add Members: Add supplier emails
4. Save

**Daily use:**
- To: "Italian Food Suppliers" (sends to all)

### 2. Use Outlook Templates

**Save your subject lines:**
1. New Email → Type subject
2. File → Save as Template
3. Name: "FDX Partnership Inquiry"

**Quick access:**
- Home → New Items → More Items → Choose Form

### 3. Schedule Strategic Sending

**Best times to send:**
- Europe: 9-10 AM CET
- Asia: 9-10 AM local time
- Americas: 9-10 AM EST

**Delay delivery:**
1. Options → Delay Delivery
2. Set date/time
3. Emails send automatically

### 4. Track Opens with Read Receipts

1. Options → Request a Read Receipt
2. When they open, you get notification
3. Update FoodXchange pipeline accordingly

## 📊 Outlook + FoodXchange Sync

### When You Send
1. FoodXchange tracks: "Email Sent"
2. Supplier moves to: "Contacted"

### When They Reply
1. See reply in Outlook

After sending batch:
1. Go to `/email/pipeline`

1. Type common phrase
2. Select text
3. Insert → Quick Parts → Save Selection

## 🆘 Common Issues

**"Emails going to spam"**
- Ask recipients to whitelist your email
- Avoid too many links
- Keep emails under 150 words

**"Copy/paste formatting issues"**
- Paste as plain text first: `Ctrl + Shift + V`
- Then format in Outlook

**"Too many suppliers to track"**
- Limit to 20-30 per day
- Focus on one region at a time
- Use pipeline view religiously
