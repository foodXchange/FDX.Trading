Getting Started with FDX Trading

Welcome to FDX Trading!

FDX Trading is your comprehensive procurement management platform. Follow these steps to get started quickly:

  1. Dashboard Overview - Start from your dashboard to see pending requests and quick actions
  2. Create Your First Request - Click "Create Request" to begin a new procurement request
  3. Browse Suppliers - Explore our verified supplier catalog
  4. Manage Products - Add products to your catalog for easy reordering
  5. Track Progress - Monitor your requests through the approval workflow
  6. Generate Reports - Access analytics and insights about your procurement

Pro Tip

Use the Command Palette (Ctrl+K or Cmd+K) to quickly navigate to any page or perform actions without using the mouse.

Features Guide

Procurement Requests

Create, manage, and track procurement requests through the entire approval workflow. Set priorities, add multiple items, and attach supporting documents.

Supplier Management

Browse verified suppliers, view catalogs, compare prices, and manage your preferred supplier list. Contact suppliers directly through the platform.

Analytics & Reports

Access real-time analytics, spending trends, and generate custom reports. Export data in multiple formats for further analysis.

User Management

Manage user accounts, set permissions, and configure approval workflows. Import users in bulk via CSV files.

Frequently Asked Questions

How do I create a new procurement request?
To create a new procurement request, click the "Create Request" button from the dashboard or navigate to the Requests page. Fill in the required information including title, description, priority, and add items. You can save as draft or submit for approval.
How can I track my request status?
You can track your request status from the "My Work" page or the main Requests page. Each request shows its current status (Draft, Pending, Approved, Rejected) and you can click on any request to see detailed timeline and approval history.
Can I export my data?
Yes! You can export data from most pages using the Export button. We support CSV, Excel, and PDF formats. For custom reports, visit the Reports page where you can create and schedule automated exports.
How do I enable dark mode?
Dark mode is automatically enabled based on your system preferences. You can also manually toggle it from the Settings page under Appearance preferences.
What browsers are supported?
FDX Trading supports all modern browsers including Chrome (90+), Firefox (88+), Safari (14+), and Edge (90+). For the best experience, we recommend using the latest version of Chrome or Edge.

API Documentation

Base URL

https://api.fdxtrading.com/v1

Authentication

All API requests require authentication using a Bearer token. Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Common Endpoints

  • GET /requests - List all procurement requests
  • POST /requests - Create a new request
  • GET /suppliers - List all suppliers
  • GET /products - List products catalog
