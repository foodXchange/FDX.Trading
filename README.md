# FoodXchange - AI-Powered Supplier Management Platform

## Overview
FoodXchange is an intelligent platform for managing food suppliers, automating communications, and optimizing purchasing decisions using AI.

## Quick Start
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Configure environment variables
4. Run backend: `python app_integrated.py`
5. Run frontend: `npm start`

## Features
- 🤖 AI-powered email analysis
- 📊 Supplier intelligence dashboard
- 📧 Automated email campaigns
- 💰 Price prediction and analytics
- 🌍 20,000+ supplier database

## Tech Stack
- Backend: Python, FastAPI, Azure OpenAI
- Frontend: React, Tailwind CSS
- Database: Azure Cosmos DB
- Email: SMTP Integration
