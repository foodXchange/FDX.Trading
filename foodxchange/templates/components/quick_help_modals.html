FDX.trading API Reference

Complete API documentation for integrating FDX.trading with your systems.

Authentication
POST /api/auth/login
  Body: { "email": "user@example.com", "password": "pass123" }
  Returns: { "token": "jwt_token", "user": {...} }

POST /api/auth/register
  Body: { "email": "user@example.com", "password": "pass123", "company": "Company Name" }
  Returns: { "success": true, "message": "Registration successful" }

GET /api/auth/logout
  Headers: { "Authorization": "Bearer jwt_token" }
  Returns: { "success": true }
Product Analysis
POST /api/analyze/product
  Headers: { "Authorization": "Bearer jwt_token" }
  Body: FormData with image file
  Returns: {
    "product_name": "Organic Honey",
    "ingredients": ["100% Pure Honey"],
    "nutritional_info": {...},
    "certifications": ["USDA Organic", "Fair Trade"]
  }
Supplier Management
GET /api/suppliers
  Query: ?category=honey&location=usa&certified=true
  Returns: { "suppliers": [...], "total": 150 }

POST /api/suppliers/rfq
  Body: { "supplier_ids": [1,2,3], "product": "Organic Honey", "quantity": 5000 }
  Returns: { "rfq_id": "RFQ-2024-001", "status": "sent" }

Troubleshooting Guide

Common issues and solutions for FDX.trading platform.

Image Analysis Issues

Problem: AI cannot read product information

Solutions:

  • Ensure image is clear and well-lit
  • Upload higher resolution images (min 1024x768)
  • Avoid blurry or angled photos
  • Try multiple angles if needed
Login Issues

Problem: Cannot access account

Solutions:

  • Check email verification status
  • Use password reset if forgotten
  • Clear browser cache and cookies
  • Contact support if account is locked
