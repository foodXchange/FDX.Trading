API Documentation

Integrate FoodXchange into Your Systems

Authentication Required

All API endpoints require authentication using API keys. Contact our support team to get your API credentials.

Getting Started

The FoodXchange API allows you to integrate our platform's functionality directly into your applications. Our RESTful API supports JSON requests and responses.

Base URL
https://api.foodxchange.com/v1/
Authentication

Include your API key in the request headers:

Authorization: Bearer YOUR_API_KEY

Product Search

GET /products/search

Search for products in our database

Parameters
Parameter Type Required Description
query string Yes Search query for products
category string No Filter by product category
country string No Filter by supplier country
limit integer No Number of results (max 100)
Example Request
GET /products/search?query=organic+tomatoes&category=vegetables&limit=20
POST /products/analyze

Analyze a product image using AI

Request Body
{ "image_url": "https://example.com/product.jpg", "analysis_type": "full" }

Supplier Management

GET /suppliers

Get list of verified suppliers

GET /suppliers/{id}

Get detailed supplier information

POST /suppliers/{id}/contact

Send message to supplier

Order Management

POST /orders

Create a new order

Request Body
{ "supplier_id": "sup_123456", "products": [ { "product_id": "prod_789", "quantity": 100, "unit": "kg" } ], "delivery_address": { "street": "123 Main St", "city": "New York", "country": "USA" } }
GET /orders

Get your order history

GET /orders/{id}/status

Track order status

Response Format

All API responses follow this format:

Success Response
{ "success": true, "data": { // Response data here }, "message": "Request successful" }
Error Response
{ "success": false, "error": { "code": "INVALID_REQUEST", "message": "The request parameters are invalid" } }

Rate Limits

  • Free Tier: 1,000 requests per hour
  • Pro Tier: 10,000 requests per hour
  • Enterprise: Custom limits available

SDKs and Libraries

We provide official SDKs for popular programming languages:

  • JavaScript/Node.js: npm install foodxchange-api
  • Python: pip install foodxchange-python
  • PHP: composer require foodxchange/api
  • Java: Maven and Gradle packages available
